# Decentralized Voting System (Algorand)

This project implements a blockchain-based decentralized voting system using the Algorand blockchain.

## Features
- Secure voting using Algorand smart contracts
- Transparent vote counting
- Tamper-proof election records

## Tech Stack
- Algorand Blockchain
- PyTeal / TEAL
- JavaScript Frontend
- AlgoSDK
